Welcome to the new blog

March 19, 2016 Jean-Sébastien Pédron

1 minute read

We finally took time to recover the content of the previous lost blog, hosted on http://blogs.freebsdish.org/graphics/ and create this new blog.

Dave Cottlehuber (dch on #freebsd-xorg) was of a great help to achieve that! He recovered the articles from the Web archives and made them available to us. He also suggested the tools we are now using to create and host this blog: GitHub pages and Hugo.

About
